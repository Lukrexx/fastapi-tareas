# API de Tareas con Autenticación 🔐

Proyecto backend desarrollado con FastAPI.

## 🚀 Funcionalidades

- CRUD de tareas
- Registro de usuarios
- Login con JWT
- Rutas protegidas

## 🛠️ Tecnologías

- FastAPI
- SQLite
- SQLAlchemy
- JWT (python-jose)

## 🔐 Autenticación

Para acceder a rutas protegidas:
1. Hacer login
2. Usar el token en los headers

## ▶️ Ejecutar

```bash
uvicorn main:app --reload
